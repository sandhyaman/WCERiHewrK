# 高校教室管理系统 python930

## 项目概述

高校教室管理系统是一个基于Python-Django框架和MySQL数据库的软件项目。该系统旨在简化和优化高校教室的预约与管理流程，通过多个功能模块实现教室资源的有效分配与使用。

## 技术栈

- **编程语言**: Python
- **开发工具**: Pycharm
- **数据库版本**: MySQL 5.7+
- **数据库工具**: Navicat、SQLyog
- **关键技术**: Python、Django、HTML、MySQL

## 功能模块

系统包含以下主要功能模块：

- 首页
- 个人中心
- 教师管理
- 学生管理
- 教室信息管理
- 教师申请管理
- 学生申请管理
- 课时表管理
- 教师取消预约管理
- 学生取消预约管理

## 系统角色

系统主要面向以下用户角色：

- 管理员：负责系统的维护与用户管理。
- 教师：可查看课时表、申请教室、取消预约等。
- 学生：可申请教室、查看课时表、取消预约等。

## 运行环境

- Python环境
- Django框架
- MySQL数据库
